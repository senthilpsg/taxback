# Taxback

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 7.2.3.

## Steps to install

1. `git clone https://github.com/senthilpsg/taxback`

2. `cd taxback`

3. `npm install`

4.  `ng serve` 

5.  Open browser and Navigate to `http://localhost:4200/`.
